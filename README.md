# Reproducibility package

This folder contains the scripts and processed files used to generate the main analyses and figures for the decapod crustacean gut microbiome meta-analysis.

## Scripts:
- 01_load_merge_and_composition.R
- 02_pca_by_organism.R
- 03_alpha_diversity.R
- 04_beta_diversity_pcoa.R
- 05_core_microbiome.R

## LotuS2:
- lotus2_example_slurm.sh
- sra_download_example.sh
- lotus2_perameters.txt

## Nb:
These scripts are intended as a clean reproducibility package derived from the main working analysis pipeline.
